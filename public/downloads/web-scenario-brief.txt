# Facilitator Brief: Claude For Web Scenario

Objective:

Use multiple sources together and push participants beyond summarization.

Recommended flow:

1. Open the market update, customer escalation, and ops dashboard together.
2. Ask Claude to produce a one-page brief.
3. Ask participants to challenge the first answer.
4. Ask for a revised recommendation with evidence and uncertainty separated.

Debrief:

- What would a leader miss if they only summarized one file at a time?
- What assumptions did Claude make?
- Which actions should stay human-owned?
